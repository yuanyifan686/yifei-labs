# Yifei Labs JD 导入扩展

从 **Boss 直聘 / 智联招聘** 岗位详情页提取 JD，并打开本地 Yifei Labs「能力诊断」页。

## 安装

1. 打开 Chrome → `chrome://extensions`
2. 开启「开发者模式」
3. 「加载已解压的扩展程序」→ 选择本目录 `extensions/jd-importer`
4. 确保本机已运行 `npm run dev`（默认 `http://localhost:3000`）

## 使用

1. 打开 Boss 或智联的**岗位详情页**
2. 点击扩展图标
3. 点「提取当前页 JD」
4. 点「打开能力诊断」
5. 在工作台粘贴/上传简历后开始诊断

## 说明

- 仅读取用户当前打开页面的可见内容，不做全站爬取
- 页面结构变更可能导致提取不完整，可手动复制 JD 到诊断页
- 生产环境把弹窗里的地址改成你的部署域名
